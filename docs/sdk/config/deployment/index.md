# Deployment Configuration

Deployment configuration objects for managing network infrastructure in Strata Cloud Manager.

## Overview

This section covers deployment services provided by the Palo Alto Networks Strata Cloud Manager SDK. Each configuration object corresponds to a resource for managing network deployments, service connections, and routing.

## Available Modules

| Module                                            | Description                                     |
|---------------------------------------------------|-------------------------------------------------|
| [Bandwidth Allocations](bandwidth_allocations.md) | Manage bandwidth allocation objects             |
| [BGP Routing](bgp_routing.md)                     | Manage BGP routing singleton configuration      |
| [Internal DNS Servers](internal_dns_servers.md)    | Manage internal DNS server objects               |
| [Network Locations](network_locations.md)         | List and retrieve network location objects       |
| [Remote Networks](remote_networks.md)             | Manage remote network configurations             |
| [Service Connections](service_connections.md)      | Manage service connection objects                 |
