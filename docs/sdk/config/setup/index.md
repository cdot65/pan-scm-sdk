# Setup Configuration

Setup configuration objects for organizing and managing resources in Strata Cloud Manager.

## Overview

This section covers setup services provided by the Palo Alto Networks Strata Cloud Manager SDK. These objects form the foundation for organizing and managing your environment.

## Available Modules

| Module                        | Description                                                    |
|-------------------------------|----------------------------------------------------------------|
| [Device](device.md)          | List, filter, and retrieve device resources                    |
| [Folder](folder.md)          | Manage folder objects for hierarchical resource organization   |
| [Label](label.md)            | Manage label objects for resource classification               |
| [Snippet](snippet.md)        | Manage reusable configuration snippets                         |
| [Variable](variable.md)      | Manage variable resources with flexible typing and containers  |
