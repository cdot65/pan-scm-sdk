# Insights Configuration

Access the Prisma Access Insights API for visibility into security posture, performance metrics, and operational health.

## Overview

The Insights configuration section provides programmatic access to alerts, metrics, and telemetry data from your Prisma Access deployment through the Palo Alto Networks Strata Cloud Manager SDK.

| Module                  | Description                                           |
|-------------------------|-------------------------------------------------------|
| [Alerts](alerts.md)     | Access and analyze security and operational alerts    |
