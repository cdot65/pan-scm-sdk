# Deployment Models

Data models for deployment resources in Strata Cloud Manager.

## Overview

The Deployment models provide structured data representations for managing deployment configurations including bandwidth allocations, BGP routing, DNS servers, network locations, remote networks, and service connections. These models handle validation of inputs and outputs when interacting with the SCM API.

## Available Models

- [Bandwidth Allocation Models](bandwidth_allocation_models.md) - Models for bandwidth allocation objects
- [BGP Routing Models](bgp_routing_models.md) - Models for BGP routing configuration
- [Internal DNS Servers Models](internal_dns_servers_models.md) - Models for internal DNS server configuration
- [Network Location Models](network_locations.md) - Models for network location objects
- [Remote Networks Models](remote_networks_models.md) - Models for remote network configuration
- [Service Connection Models](service_connections_models.md) - Models for service connection configuration
